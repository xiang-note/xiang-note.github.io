Laravel 教學資源目錄

2019/4/4 posted in  laravel comments

自 2013 年末接觸 Laravel 開始，就被優雅的寫法及架構設計所吸引，至今從零到有開發過許多專案，學習並一同成長至今，也很高興看見了 Laravel 臺灣 在臺灣技術圈的蓬勃發展。

這些年來累積不少的經驗，希望透過筆記整理下來，讓對 Laravel 有興趣的人有更多的瞭解。

導讀

其實官方文件 Laravel Document 已經寫得相當詳細。
若對英文不熟悉的人，網路上也有許多中文翻譯及手把手教學的資源。

因此本系列筆記的重點會著重於

  • 自身的理解來描述概念
  • 開發上常用技巧或語法
  • 研究 source code 才知道的用法
  • 踩過的雷，掉過的坑
  • 不限於 Laravel，有關的也會提及
  • 想到什麼寫什麼

所以此系列不適合

  • 想快速上手實戰 Laravel
  • 系統性的學習
